# footy.ai

> Die digitale Plattform für Fußballtrainer.

Eine vollständige App für die tägliche Arbeit eines Vereinstrainers — Anwesenheits­verwaltung, Aufstellungen, Saisonplanung, Statistiken und Kommunikation in einer einzigen Anwendung.

🔗 **Live**: [fc-frohlinde.github.io](https://fc-frohlinde.github.io)

## Funktionen

- 📋 **Anwesenheits­verwaltung** — Trainings und Spiele, mit Anwesenheits­status und Entschuldigungen
- ⚽ **Aufstellungs­planer** — Drag-&-Drop-Formationen für Spiele und ganze Saisons
- 👥 **Saisonkader** — Spieler­verwaltung, Bestätigungen, Anmeldungs­status
- 📊 **Statistik & Dashboard** — Detaillierte Auswertungen pro Spieler und Mannschaft
- 📅 **Kalender** — Trainings, Spiele und eigene Termine
- 💬 **Trainer-Kommunikation** — Nachrichten zwischen Trainern, Push-Benachrichtigungen
- 🔐 **Sicherheit** — Passwort/Passkey-Anmeldung, kritische Aktionen mit Bestätigung
- ☁ **Cloud-Sync** — Echtzeit-Synchronisation über Firebase

## Technik

- Single-file HTML/JavaScript application
- Firebase Realtime Database (Synchronisation)
- Firebase Authentication & Storage
- Progressive Web App (installierbar)
- Keine externen Abhängigkeiten zur Laufzeit
